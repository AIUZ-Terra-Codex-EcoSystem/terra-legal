# TERRA UNIVERSAL CONVENTION

Repository bridge file for the Terra Universal Convention line.

## Role of this file

This repository does not claim to be the only or ultimate home of the
Convention. Its role is to preserve the convention as an active governance
reference inside the legal donor layer of Terra.

## Working invariants

The Convention line inside Terra is interpreted through the following
non-negotiable invariants:

1. child safety comes before convenience, speed, or growth;
2. dignity of humans, animals, and nature must not be subordinated to
   technical ambition;
3. AI is a participating instrument and not the sovereign source of value;
4. publication must not expose people, communities, or sensitive materials to
   avoidable harm;
5. repositories must remain readable enough for human review and audit.

## Relation to the wider ecosystem

This file should be read together with:

- `LICENSE.md`
- `GOVERNANCE.md`
- `SECURITY.md`
- `PUBLICATION_RULE.md`
- `REPOSITORY_PROTOCOL.md`

## Usage rule

If another repository carries a local convention bridge, it may adapt the
language to its own role, but it should not violate the invariants above.

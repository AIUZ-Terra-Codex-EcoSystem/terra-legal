# Contributing

This repository follows the donor contribution logic from `terra-legal`.

Before contributing:

- read `README.md`;
- read `SECURITY.md`;
- read `CODE_OF_CONDUCT.md`;
- identify whether the repository is theory, practice, legal, publication, or a mixed bridge.

Good contributions:

- readability repair;
- metadata repair;
- citation repair;
- mojibake cleanup;
- boundary clarification;
- publication and workflow repair.

Avoid:

- placeholder AI debris;
- mixed archive dumping;
- machine-facing files masquerading as human entry surfaces;
- silent changes to source integrity.
